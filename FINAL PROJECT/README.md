# MultiBot - Многофункциональный Телеграм-бот

## Описание
MultiBot — это Телеграм-бот, который показывает погоду, шутки и помогает вести список задач. Разработан как финальный проект по курсу Python.

## Возможности
- 🌤️ **Погода**: Получение погоды для любого города
- 😂 **Шутки**: Случайные шутки
- 📋 **Список задач**: Добавление, выполнение и удаление задач
- 💾 **База данных**: SQLite хранит все задачи

## Используемые технологии
- Python 3.9+
- python-telegram-bot
- SQLite
- OpenWeatherMap API
- JokeAPI

## Структура проекта
final_project/
├── bot.py # Главный файл — запускает бота
├── config.py # API-ключи и настройки
├── db.py # Работа с базой данных
├── handlers.py # Обработчики команд
├── utils.py # Вспомогательные функции
├── requirements.txt # Зависимости
└── README.md # Этот файл


## Установка и запуск

1. Установите зависимости
pip install -r requirements.txt

2. Настройте API-ключи
Получите токен бота у @BotFather

Получите API-ключ OpenWeatherMap

Обновите config.py своими ключами

3. Запустите бота
python bot.py

Команды
Команда-            Описание
/start-	            Главное меню
/help-	            Список команд
/weather <город>-	Погода
/joke-	            Шутка
/todo-	            Список задач
/addtodo <задача>-	Добавить задачу
/done <id>-	        Выполнить задачу
/deltodo <id>-	    Удалить задачу
/about-	            О боте
